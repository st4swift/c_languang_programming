## [CS自学指南 (csdiy.wiki)](https://csdiy.wiki)

由国内清华大学等顶尖高校学长维护的开源平台。它汇总了国内外最高质量的计算机课程（如著名的 CSAPP：深入理解计算机系统 课程）。如果你未来想用C语言走向系统级开发、嵌入式或操作系统开发，按照这个指南去刷课，含金量极高。
